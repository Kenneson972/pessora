get-stripe-member

Retourne : statut, next billing date, cancel_at_period_end, moyen de paiement (brand + last4)
cancel-stripe-subscription
Input : stripe_subscription_id
Action : cancel_at_period_end: true → mise à jour Stripe + DB
admin-portal-session
Input : stripe_customer_id
Retourne : URL portail Stripe → ouvre dans nouvel onglet
